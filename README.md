一般的操作流程：

1. git clone 到本地
``` 
git clone git@github.com:pengshiqi/Visual-computing-theory-report.git 
``` 

2. 做出自己的修改

3. 
```  
git add main.tex 
``` 

4. 
```  
git commit -m 'Commit Message'
``` 
Commit Message 是你的commit说明。

5. 
```  
git pull
``` 
如果有conflict，就解决之。

6. 
```  
git push origin master
```
